# ShareX

## Desktop File Sharing Application

ShareX is a JavaFX-based desktop application that enables users to securely send and receive files over a local network with real-time progress updates.

## Features

- User Registration & Login
- Secure File Transfer
- Send Multiple Files
- Real-time Progress Bar
- Accept/Reject Incoming Files
- Save File to Custom Location
- JavaFX GUI
- Socket Programming
- MVC Architecture

## Technologies Used

- Java 21
- JavaFX
- Maven
- Firebase Authentication
- Firebase Realtime Database
- Socket Programming
- CSS
